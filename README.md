# Saldopilot

Administrador personal de ingresos, gastos, presupuestos, gastos fijos y objetivos de ahorro.

## Ejecutar

```bash
npm install
npm run dev
```

## Publicar en GitHub Pages

La configuración de Vite usa `base: './'`, por lo que el resultado de `npm run build` queda listo en `dist/` para publicarse como sitio estático.
